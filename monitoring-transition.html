Monitoring transition

Manual - Automated - Distributed

Saurabh Hirani / @sphirani

Why?

Manual monitoring

  • Better than no monitoring
  • New hosts?
  • Decomissioned hosts?
  • Remote operations?
  • Blocked on wizards



Automated monitoring

  • No manual intervention
  • Auto generated configs from host inventories
  • REST APIs
  • Single point of failure



Distributed monitoring

  • No single monitoring server
  • Distributed checks
  • Horizontal scalability
