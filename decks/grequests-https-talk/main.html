The curious case of slow/fast grequests code

saurabh.hirani@gmail.com / @sphirani

Use case

  • Daily reporting tool: 1000s of HTTP GETs
  • Runtime: 15-20 mins (serial requests)
  • Migrate from AWS EC2 to AWS Lambda
  • Other requirements: Migrate to HTTPS + Python3

Problem

  • Constraint: AWS Lambda 15 mins max runtime
  • Solution: Convert serial GETs to concurrent
  • requests => grequests
  • Hidden trap: Execution speed varies.
  • Depends on: Other Python modules installed on system.

requests module flow

  • Client: Open a connection to the server.
  • Client: Send GET request.
  • Client: Wait for server to respond.
  • Server: Sends response.
  • Client: Repeat same activity for all URLs.

grequests module flow

  • Client: Open a connection to the server.
  • Client: Send GET request.
  • Client: I am not waiting, here's a callback - fire it when the server responds.
  • Client: Repeat same activity for all URLs.
  • Client: Waits for response(s).
  • Server: Sends responses (unordered).

Why pyopenssl?

  • grequests = gevent + requests
  • requests optionally includes pyopenssl - for SNI support.
  • stdlib ssl did not provide SNI in Python 2.7.8 and older.
  • pyopenssl monkey patches stdlib ssl to add SNI.
  • stdlib in Python 2.7.9+ - ssl.HAS_SNI flag is True.

How is pyopenssl making the code slow?

  • pyopenssl changes program flow and re-patches socket
  • Demo re-patching
  • grequests -> monkey_patch -> requests -> pyopenssl -> monkey_patch
  • Stage-1 socket class
  • Stage-2 socket class

Solution

  • Uninstall pyopenssl
  • OR
  • Use gevent-openssl
  • gevent-openssl = pyopenssl compatible with gevent

Stage-3 - Fix-1

  • Stage-2 + gevent-openssl
  • Demo
  • Python 2.7 fast
  • Python 3.7 slow

Why?

  • Python2.7 uses recv
  • Python3.7 uses recv_into
  • gevent-openssl does not override recv_into

Stage-4 - Fix-2

  • Patch gevent-openssl to add recv_into
  • Before patching demo
  • After patching demo
  • PR to fix the issue

Recap

  • Applies only to HTTPS
  • grequests + pyopenssl = slow for Python2.7/Python3.7
  • grequests - pyopenssl = fast for Python2.7/Python3.7
  • grequests + pyopenssl + gevent-openssl = fast for Python2.7, slow for Python 3.7
  • grequests + pyopenssl + gevent-openssl + patch = fast for Pytho2.7/Python3.7
